## [1.0.0] - 2025-12-04

### 🎉 首次发布

一个完整的、高性能的 Android Logcat 日志查看工具。

### ✨ 新功能

#### 设备管理
- 自动检测已连接的 Android 设备和模拟器
- 显示设备列表（名称、序列号、状态）
- 启动时自动连接第一个可用设备
- 区分真机和模拟器

#### 日志显示
- 实时 logcat 日志流
- 日志级别颜色区分 (V/D/I/W/E/A)
- 虚拟滚动支持百万级日志
- 可拖拽调整列宽
- 消息自动换行开关
- 自动滚动到最新日志
- 暂停/恢复日志流

#### 列配置
- 时间戳（支持日期-时间、时间、时间戳格式切换）
- PID/TID（可选合并显示）
- 包名和进程名
- 日志级别
- TAG
- 支持隐藏重复值（TAG、包名、进程名）

#### 过滤功能
- 完全兼容 Android Studio Logcat 过滤语法
- 三种匹配模式：包含 `tag:` / 精确 `tag=:` / 正则 `tag~:`
- 支持排除过滤 `-tag:`
- AND（空格）和 OR（|）组合
- 级别过滤 `level:INFO`
- 时间过滤 `age:5m`
- 特殊过滤 `is:crash` `is:stacktrace`
- Match Case 区分大小写开关

#### 过滤历史
- 自动保存最近 20 条过滤记录
- 收藏常用过滤条件（无上限）
- 输入时显示语法提示和历史匹配
- 底部显示使用 Tips

#### 导入导出
- 导出为 `.logcat` 格式（Android Studio 兼容）
- 导入 `.logcat` 文件
- 导入纯文本日志（自动格式检测）
- 导入后显示文件图标和文件名

#### 界面
- 深色/浅色主题切换
- 左侧垂直工具栏（参考 Android Studio）
- 底部状态栏显示统计信息
- 设置面板可视化配置

### 🏗️ 技术架构
- Tauri 2.0 跨平台框架
- Rust 后端（ADB 通信、日志解析）
- React 18 + TypeScript 前端
- Tailwind CSS 样式
- Zustand 状态管理
- @tanstack/react-virtual 虚拟滚动
